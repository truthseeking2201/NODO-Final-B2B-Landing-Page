## ğŸ“¦ Installation & Setup

To run this project locally, follow these steps:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/truthseeking2201/NODO-Final-B2B-Landing-Page.git
    cd NODO-Final-B2B-Landing-Page
    ```

